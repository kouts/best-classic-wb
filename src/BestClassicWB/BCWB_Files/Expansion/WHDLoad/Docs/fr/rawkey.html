Les codes boutons

A partir de la version 16.8 de WHDLoad, il est possible de quitter WHDLoad avec les boutons de la souris, joystick ou joypad. Pour cela, il y a des rawkeys spéciaux listés dans le tableau ci-dessous. Ces rawkeys ne fonctionnent seulement avec l'option QuitKey, pas avec d'autres options key.
Pour que WHDLoad puisse détecter qu'un bouton a été pressé, l'option NoVBRMove ne doit pas être utilisée, le CPU doit être un 68010 au moins et le programme installé doit avoir l'interruption vertical blank active.
Les rawkeys ci-dessous sont représentés avec valeurs hexadécimal ou décimal.
En utilisant les boutons d'un joypad ($103..$10a/$113..$11a) comme un QuitKey, il sera impossible au programme installé de détecter/tester/utiliser un joypad. Celui-ci pensera qu'un joystick est connecté. A cause des limites du Hardware, le joypad peut seulement être testé une fois pendant un 1/50ième de seconde. Alors ne l'utilisez pas pour les jeux qui nécessitent un joypad.
L'option QuitKey peut être utilisée en pressant Reverse, Forward et Play ensemble et est supportée à partir des versions 18.4 de WHDLoad.

Dans les versions 16.4 à 16.7 de WHDLoad, il était possible d'utiliser un QuitKey de $73 pour quitter en pressant le bouton gauche de la souris. Ceci ne marche plus dans les versions 16.8 et supérieures. Le nouveau code rawkey est $100 pour cela.
