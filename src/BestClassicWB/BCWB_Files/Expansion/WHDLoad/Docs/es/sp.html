SP * Salvar Pantalla

Es una herramienta para extraer una pantalla del fichero de volcado creado por WHDload.

Instalación

SP se encuentra solamente en el archivo DEV, no en el archivo USR
cópielo a C: o una ubicación similar

Uso

Primero debe crearse un archivo de volcado. Inicie el programa instalado desde el cual quiere extraer la pantalla. Debe ser ejecutado con la opción Snoop/S activada. También deben estar configuradas Expert/S y DebugKey/K/N. Si se muestra la pantalla que desea extraer presione la tecla de DebugKey. WHDLoad escribirá entonces el archivo de volcado. Entre al directorio donde esta ubicado el archivo de volcado usando una Shell/CLI. Ahora invoque SP con el nombre del archivo a grabar como argumento. SP cargará el volcado e intentará salvar la pantalla. So ha configurado la opción CoreDumpPath no necesitará ingresar al directorio del archivo de volcado de memoria porque SP lee el archivo de configuración global de WHDLoad para encontrarlo.

Opciones

OutputFile/A,Cop/K,CS=CopStop/K,Width/K,Height/K,con0/K,mod1/K,mod2/K,pt1/K,pt2/K,pt3/K,pt4/K,NoCopLst/S
OutputFile/A el nombre con el cual grabar la pantalla
Cop/K configura la dirección de inicio de la primer copperlist a interpretar, si Ud. desea sobreescribir la dirección contenida en el archivo de volcado
CS=CopStop/K esta opción especifica la dirección en la cual terminar de interpretar la copperlist. Debe ser usada si la pantalla esta construida a partir de mas de una imagen picture
Width/K especifica el ancho de la imagen a salvar, no afectara los valores de modulo!
Height/K especifica la altura de la imagen a salvar
con0/K sobreescribe el registro bplcon0 contenido en el archivo de volcado
mod1/K,mod2/K sobreescribe el registro bplmod1/bplmod2 contenido en el archivo de volcado
pt1/K,pt2/K,pt3/K,pt4/K   sobreescribe el registro bplpt1,2,3,4 contenido en el archivo de volcado
NoCopLst/S desactiva el volcado de la copperlist
