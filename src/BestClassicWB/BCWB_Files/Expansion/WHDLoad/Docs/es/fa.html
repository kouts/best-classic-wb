FindAccess * Encontrar Accesos a una ubicación de memoria

Esta herramienta cargara el fichero especificado y buscara en el mismo los accesos a una dirección de memoria especificada. Encontrara todas las referencias absolutas y relativas de 8-bits, 16-bits y 32-bits. Ha sido escrito para analizar los ficheros de volcado de memoria. Es similar al comando Freezer (FA) pero no desensamblara el acceso, solo mostrara el mismo como un volcado en hexadecimal.

Instalación

FindAccess solo esta presente en el archivo DEV, no en el archivo USR
cópielo al C: o una ubicación similar

Uso

Yo uso el siguiente alias (agregado a mi S:Shell-Startup) para buscar en el último volcado de memoria escrito por whdload:

alias fa FindAccess C:.whdl_memory

de tal forma que puedo abrir una consola (Shell) y escribir, por ejemplo:

fa $15000

y obtendré la salida:

FindAccess 1.2 (17-Jan-99 14:45:18) by Bert Jahn
loading file Workbench:CE/.whdl_memory
scanning accesses to $15000 (86016) file: $0-$80000
Relative Word at $ 14240 -> 06C00000 07C00000 0DC0 00000CC0 000004C0

Opciones

FILE/A,ADDRESS/A,ORG
FILE/A el nombre del fichero en el cual buscar
ADDRESS/A   la dirección a buscar, pueden usarse expresiones simples como "$5000" = "20480" = " -$Ff +-33+ 19512+$4e8" (prefijo "$" para números en hexadecimal, operadores soportados "+" y "-", unario y binario)
ORG la dirección lógica de inicio del archivo, si no se especifica se usa $0, Ud. puede usar el mismo tipo de expresiones simples que con ADDRESS/A
