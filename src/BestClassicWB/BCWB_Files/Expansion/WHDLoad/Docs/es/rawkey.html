Los códigos de botón

Comenzando con WHDLoad versión 16.8 también es posible salir de WHDLoad usando un botón del ratón, joystick o joypad. Para ello hay códigos de teclado especiales definidos en la tabla debajo. Estos códigos de teclado solo funcionan con la opción QuitKey, no con ninguna otra opción de teclado.
Para hacer que WHDLoad sea capaz de detectar cuando un botón es presionado no debe usarse la opción NoVBRMove, el CPU debe ser como mínimo un 68010 y el programa instalado debe tener la interrupción de blanqueo vertical activada.
Los códigos de teclado se listan en notación hexadecimal y decimal.
El uso de botones desde un joypad ($103..$10a/$113..$11a) como la QuitKey hará imposible que el programa instalado detecte/chequee/use el joypad. El programa instalado creerá que un joystick está conectado. Esto se debe a algunas limitaciones de hardware que implican que el joypad solo puede ser chequeado una vez por cada 1/50 de segundo. Por lo tanto no utilice esta opción para juegos que requieren utilizar un joypad.
La función QuitKey utilizada al presionar Reversa, Adelante y Jugar al mismo tiempo comenzó a estar soportada a partir de la version 18.4 de WHDLoad.
