CRC16 * Cálculo de la suma de verificación CRC16

Este programa calcula el CRC-16 de la porción especificada de un fichero (o el fichero completo). La idea es básicamente su uso en la automatización de la instalación con el fin de identificar con que versión de un fichero dado están trabajando para instalar diferentes Esclavos, etc. El valor retornado es el mismo que el CRC-16 calculado por resload_CRC16 el cual será útil también durante el desarrollo de Esclavos.

Instalación

cópielo a C: o una ubicación similar

Opciones

FILE/A,OFFSET/K/N,LENGTH/K/N
FILE/A fichero sobre el cual calcular la suma de verificación
ejemplo "S:Startup-Sequence"
OFFSET/K/N   cantidad de bytes a saltar desde el principio del fichero
valor por defecto: principio del fichero
ejemplo "OFFSET=11264" para comenzar en la pista 2 de una imagen de disco estándar
LENGTH/K/N cantidad de bytes sobre los cuales calcular el CRC
valor por defecto: fin del fichero
ejemplo "LENGTH=1024"

Código Retornado

El valor CRC16 es retornado como el código de finalización del programa.
