CRC16 * Calcola il Checksum CRC16

Questo programma calcola il CRC16 di una specifica porzione di un file (o dell'intero file). L'idea di base è di utilizzarlo negli script di installazione che hanno bisogno di identificare con quale versione di un file stanno lavorando per installare Slave differenti etc. Il valore che ritorna è lo stesso del CRC16 calcolato da resload_CRC16 che può anche tornare utile durante lo sviluppo di uno Slave.

Installazione

copialo in C: o in un posto simile

Opzioni

FILE/A,OFFSET/K/N,LENGTH/K/N
FILE/A file dal quale calcolare il checksum
esempio "S:Startup-Sequence"
OFFSET/K/N   offset di inzio nel file
di default è l'inizio del file
esempio "OFFSET=11264" per cominciare alla traccia 2 di un'immagine disco standard
LENGTH/K/N lunghezza in byte per la quale calcolare il checksum
di default è la fine del file
esempio "LENGTH=1024"

Return code

Il CRC16 viene ritornato come exit code del programma.
