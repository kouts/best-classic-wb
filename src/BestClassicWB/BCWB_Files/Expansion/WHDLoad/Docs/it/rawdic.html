RawDIC * Creatore di Immagini Disco non standard

RawDIC è la scelta migliore per gli installatori che usano WHDLoad per creare file o immagini disco dai dischi originali che utilizzano un loro formato proprietario. Per saperne di più riguardo a RawDIC, leggi la sua documentazione, che può essere trovata nel pacchetto DEV di WHDLoad.
