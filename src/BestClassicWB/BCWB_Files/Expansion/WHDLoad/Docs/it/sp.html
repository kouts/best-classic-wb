SP * Save Picture (Salva Immagine)

Un tool per estrarre un'immagine da un file di dump creato da WHDLoad.

Installazione

SP è contenuto solo nell'archivio DEV, non in quello USR
copialo in C: o in un posto simile

Utilizzo

Prima di tutto bisogna creare un file di dump. Poi avvia il programma da cui estrarre l'immagine. Deve essere avviato con l'opzione Snoop/S abilitata. Anche Expert/S e DebugKey/K/N devono essere abilitate. Se l'immagine che vuoi estrarre appare premi il DebugKey. WHDLoad scriverà il file di dump. Vai nella directory del dump con una Shell. Ora avvia SP con il nome del file da salvare come argomento. SP caricherà il file di dump e tenterà di salvare l'immagine. Se hai selezionato l'opzione CoreDumpPath non devi inserire la directory, SP leggerà il file di configurazione di WHDLoad per trovarlo.

Opzioni

OutputFile/A,Cop/K,CS=CopStop/K,Width/K,Height/K,con0/K,mod1/K,mod2/K,pt1/K,pt2/K,pt3/K,pt4/K,NoCopLst/S
OutputFile/A il nome del file dell'immagine da salvare
Cop/K imposta l'indirizzo di partenza della prima copperlist da interpretare, se vuoi sovrascrivere l'indirizzo contenuto nel dump
CS=CopStop/K questa opzione specifica l'indirizzo di fine interpretazione della copperlist. Deve essere utilizzato se lo schermo è formato da più di una immagine
Width/K specifica la larghezza dell'immagine da salvare, non interferisce con i valori di modulo!
Height/K specifica l'altezza dell'immagine da salvare
con0/K sovrascrive il registro bplcon0 contenuto nel dump
mod1/K,mod2/K sovrascrive il registro bplmod1/bplmod2 contenuto nel dump
pt1/K,pt2/K,pt3/K,pt4/K   sovrascrive il registro bplpt1,2,3,4 contenuto nel dump
NoCopLst/S disabilita il dump della copperlist
