Installazione

Il modo più semplice per installare WHDLoad ed i suoi strumenti è quello di utilizzare l'apposito script di installazione fornito.

La cosa più importante è che WHDLoad ed i suoi strumenti devono essere accessibili senza specificare un path (percorso). La loro destinazione raccomandata è "C:". WHDLoad può anche essere installato nella stessa directory dove il programma da installare è presente (nota che nelle Icone Progetto degli Esempi viene utilizzato solo "WHDLoad" senza alcun path).

Gli strumenti "DIC", "RawDIC" e "Patcher" dovrebbero essere sempre installati perché vengono richiesti dalla maggior parte degli installatori durante la procedura d'installazione. Non sono richiesti durante l'esecuzione, dopo che l'installazione è stata completata.
