InstallBB * Installieren von Bootblöcken

InstallBB kann den Bootblock von Disktette lesen oder ihn auf Diskette schreiben. Dabei wird der Bootblock als ausführbare Datei gelesen oder geschrieben. Beim Schreiben wird der Dateisystemtyp nicht verändert indem erst der alte Bootblock gelesen wird und dessen DosType (DOS0, DOS1 ...) in den neuen eingesetzt wird. Die Bootblockprüfsumme wird automatisch berechnet.

Installation

InstallBB ist nur im DEV, nicht im USR Archiv enthalten.
Kopiere das Programm nach C: oder einem ähnlichen Platz.

Optionen

PROGRAM/A,UNIT/K/N,WRITE/S
PROGRAM/A   der Name der ausführbaren Datei welche gelesen oder geschrieben wird, muss angegeben werden
UNIT/K/N die Unit Nummer des Trackdisk-Gerätes (Laufwerksnummer)
voreingestellt ist 0 (DF0:)
WRITE/S Schalter der entscheidet, ob der Bootblock gelesen oder geschrieben wird, wenn nicht gesetzt, wird der Bootblock von der Diskette gelesen und als Datei gespeichert, ansonsten umgekehrt
