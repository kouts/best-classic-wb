RawDIC * Erzeugen von nichtstandard Diskettenabbildern

RawDIC ist das bevorzugte Programm für WHDLoad Installationen zur Extrahierung von Dateien oder der Erzeugung von Diskettenabbildern von original Disketten, welche mit einem eigenen nicht üblichen Diskettenformat beschrieben worden sind. Um mehr über RawDIC zu erfahren, bitte die zugehörige Dokumentation konsultieren, welche sich im DEV Paket von WHDLoad befindet.
